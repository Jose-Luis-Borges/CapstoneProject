Hi!

My name is José Luís Borges and I'm a long-time software engineer that until very recently had no web development experience.

This summer I decided I wanted to learn enough to eventually become a full stack web developer. My journey started naturally with learning front-end web development and for that I chose the University of Michigan Web Design for Everybody online specialization on Coursera.

While doing this specialization, I found out some other online (and offline) resources — books, courses, videos — that I consider particularly appropriate for those starting their journey in the road to web development mastery.

This project presents a compilation of learning resources I selected for their relevance, quality and availability. Most of them are either free or inexpensive. This is my personal collection and I surely would like to know what others have found out.

I plan to have a professionally hosted version of this project soon, with substantially more content and topics. The current version is just the first step towards that goal.

I hope you find it useful as it is.

All the best,

José
