2013 |  184 pages |  4.3 | by  Jennifer Robbins
Handy quick reference to HTML5 elements and attributes from a pioneer web designer. Recommended if you still prefer printed media to online references or find them complementary.

CSS

CSS: The Definitive Guide: Visual Presentation for the Web, 4th Edition

2017 |  1090 pages |  4.8 | by  Eric A. Meyer and Estelle Weyl
This is a comprehensive guide to CSS implementation, covering: selectors, specificity, and the cascade; values, units, and text properties; padding, borders, outlines, and margins; colors, background, and gradients; floats and positioning tricks; flexible box layout; the new grid layout system; 2D and 3D transforms, transitions and animation; filters, blending, clipping and masking; media and feature queries. 😓

JavaScript

You Don't Know JS: Up & Going

2015 |  88 pages |  4.6 | by  Kyle Simpson
The first of a six book series on programming in JavaScript. We found that for a thorough understanding of JS these books are amongst the best resources. Focus on the language itself, not its use in the context of making interactive web pages. Free version available here.

Eloquent JavaScript, 3rd Edition: A Modern Introduction to Programming

2018 |  472 pages |  4.3 | by  Marijn Haverbeke
Another introduction to programming in JavaScript. It starts slowly but takes up speed in a short time. Not easy and for that reason not recommended for beginners but excellent for seasoned programmers. There is a free electronic version here.
